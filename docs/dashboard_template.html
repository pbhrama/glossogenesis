Pidgin vs. baseline negotiation: results

Two Qwen-powered agents negotiate the same tasks two ways — coining and sealing their own shared vocabulary (pidgin) vs. working from a full schema known upfront (baseline). This page tracks how fast each approach converges, and whether the pidgin's vocabulary cost actually disappears over time.

Rounds to converge, per task

Total negotiation rounds, pidgin vs. baseline. Round count also reflects how contentious each task's content is — it isn't purely a vocabulary-cost measure.

The learning curve: clarification rounds per task

Rounds the pidgin arm spent asking "what does that mean?" before a coined term could be used freely. This is the number that should fall as the shared dictionary matures.

Shared dictionary

Terms coined mid-negotiation and sealed until explicitly clarified, then reused for free in every task after.
